# Image Processing From Scratch

## Description
Custom image processing toolbox developed from scratch for pixel-level manipulations and matrix transformations.

## Project Purpose & Educational Design
This is an **educational framework** designed to implement core computer vision and image processing algorithms without relying on heavy external third-party functions.
It focuses on mastering the underlying matrix mathematics and raw pixel transformations, combined with a modern UI using CustomTkinter. 

*Note: Since this tool was built as part of an educational research and development phase to study algorithms deeply,
local variable naming and UI localization follow the target environment's native language, while maintaining global code standards for the underlying mathematical logic.*

## Key Features Implemented from Scratch
* **Custom Grayscale Conversion:** Implemented using raw pixel luminosity formulas instead of library shortcuts.
* **Custom Blurring Operations:** Convolution-based box/mean filter matrix operations built from scratch.
* **Adaptive Thresholding:** Dynamic pixel-level local neighborhood analysis.
* **Custom RGB to HSV Space Transformation:** Mathematical color space conversion engine developed from the ground up.
* **Advanced Filters:** Sobel edge detection, Gauss convolution, and Salt & Pepper noise generation algorithms.
* **UI/UX Design:** Dark-mode supported modern graphical interface powered by CustomTkinter.
